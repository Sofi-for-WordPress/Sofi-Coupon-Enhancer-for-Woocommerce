Enhance coupon code management in WooCommerce with the Sofi Coupon Enhancer. Display coupon orders, correct duplicated coupon counts, and add a coupon column to the order listing table.

== Description ==

The Sofi Coupon Enhancer Plugin is a robust solution designed to enhance the functionality of your WooCommerce store. With a focus on improving coupon and order management, the plugin offers an array of features that streamline the process of monitoring and managing discounts and their related orders.

== Installation ==

1. Download the plugin ZIP file from the provided repository link.
2. Log in to your WordPress admin panel.
3. Navigate to Plugins > Add New.
4. Click the Upload Plugin button.
5. Choose the downloaded ZIP file and click Install Now.
6. Activate the plugin through the 'Plugins' menu in WordPress.
